Grundsatz

Diese Website verwendet keine Cookies, kein Tracking und keine Analyse-Tools von Drittanbietern. Es werden keine personenbezogenen Daten erhoben oder gespeichert.

Lokaler Browserspeicher (localStorage)

Die Website speichert ausschliesslich folgende Einstellungen lokal im Browser:

  • mk_theme – Farbschema-Präferenz (Hell / Dunkel)
  • mk_lang – Sprachpräferenz (DE / EN)

Diese Daten verlassen den Browser nicht und werden nicht an Server übertragen. Sie können jederzeit über die Browsereinstellungen gelöscht werden.

Externe Datenquellen

Die redaktionellen Inhalte dieser Website werden wöchentlich automatisiert aus öffentlich zugänglichen RSS-Feeds der folgenden Quellen aggregiert und mit KI-Unterstützung aufbereitet:

  • FINMA – Eidg. Finanzmarktaufsicht · finma.ch
  • Schweizerische Nationalbank (SNB) · snb.ch
  • Fintech News Switzerland · fintechnews.ch
  • BankingHub · bankinghub.de
  • Netzwoche · netzwoche.ch
  • HSLU IFZ Retail Banking Blog · blog.hslu.ch/retailbanking

Zur Anzeige von Marktdaten ruft die Website zudem folgende Dienste ab:

  • GitHub (raw.githubusercontent.com) – Wöchentliche Digest-Daten (JSON). Betreiber: GitHub, Inc., San Francisco, USA. Datenschutz GitHub
  • CoinGecko API (api.coingecko.com) – Bitcoin/CHF-Kursdaten. Betreiber: Gecko Labs Pte. Ltd., Singapur. Datenschutz CoinGecko
  • Yahoo Finance (query1.finance.yahoo.com) – SMI/SPI-Indexdaten. Betreiber: Yahoo Inc., USA. Datenschutz Yahoo

Bei diesen Abrufen wird die IP-Adresse des Nutzers technisch bedingt an die jeweiligen Server übermittelt. Die Website hat keinen Einfluss auf die Datenverarbeitung dieser Drittanbieter.

Principle

This website uses no cookies, no tracking, and no third-party analytics tools. No personal data is collected or stored.

Local Browser Storage (localStorage)

The website stores only the following preferences locally in the browser:

  • mk_theme – Colour scheme preference (Light / Dark)
  • mk_lang – Language preference (DE / EN)

This data does not leave the browser and is not transmitted to any server. It can be deleted at any time via browser settings.

External Data Sources

The editorial content of this website is aggregated weekly from publicly accessible RSS feeds of the following sources and prepared with AI assistance:

  • FINMA – Swiss Financial Market Supervisory Authority · finma.ch
  • Swiss National Bank (SNB) · snb.ch
  • Fintech News Switzerland · fintechnews.ch
  • BankingHub · bankinghub.de
  • Netzwoche · netzwoche.ch
  • HSLU IFZ Retail Banking Blog · blog.hslu.ch/retailbanking

For displaying market data, the website also fetches from the following services:

  • GitHub (raw.githubusercontent.com) – Weekly digest data (JSON). Operator: GitHub, Inc., San Francisco, USA. GitHub Privacy
  • CoinGecko API (api.coingecko.com) – Bitcoin/CHF price data. Operator: Gecko Labs Pte. Ltd., Singapore. CoinGecko Privacy
  • Yahoo Finance (query1.finance.yahoo.com) – SMI/SPI index data. Operator: Yahoo Inc., USA. Yahoo Privacy

When fetching this data, your IP address is technically transmitted to the respective servers. This website has no influence over the data processing of these third-party providers.
